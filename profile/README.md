## Tentang Kami

**VivaJauh** adalah platform manajemen koperasi yang dibangun khusus untuk **koperasi peternakan** di wilayah 3T (Terdepan, Terluar, Tertinggal) Indonesia.

Kami mengatasi tantangan nyata dalam pengelolaan koperasi pedesaan, mulai dari pengumpulan data secara offline di daerah dengan konektivitas terbatas hingga distribusi pupuk bersubsidi yang transparan, dengan **pengalaman mobile offline-first** yang melakukan sinkronisasi secara mulus saat koneksi tersedia.

Platform kami membantu koperasi untuk:
- 📋 Mengelola catatan ternak, inventaris pakan, dan simpanan anggota
- 💰 Memproses pengajuan pinjaman dengan penilaian risiko berbasis AI
- 🌿 Melacak distribusi pupuk bersubsidi pemerintah (Pupuk Bersama)
- 🔄 Bekerja secara offline-first dan sinkronisasi saat terhubung
- 📊 Menghasilkan laporan yang transparan dan dapat diaudit untuk kepatuhan regulasi

## Tech Stack

| Lapisan | Teknologi |
|---|---|
| **Backend** | Node.js · Express 5 · TypeScript · Prisma · PostgreSQL |
| **Mobile** | Flutter · Dart · SQLite · BLoC/Cubit |
| **AI** | Google Gemini API · Rule-based fallback |
| **Arsitektur** | Domain-Driven Design · Offline-First · Role-Based Access Control |
